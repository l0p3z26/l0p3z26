## 👤 Sobre mí

Soy estudiante de **Sistemas Microinformáticos y Redes** en Madrid. Combino mi formación técnica en redes y sistemas con desarrollo web, bots de Discord, automatización e integración de IA generativa.

En este perfil de GitHub publico y desarrollo:

* 🎨 Prototipos web y portfolios con enfoque responsive.
* 🤖 Aplicaciones con IA, APIs externas y automatización.
* 💬 Bots de Discord con Python, SQLite, moderación, tickets, logs y arquitectura modular.
* 🛠️ Scripts y herramientas para productividad, sistemas y pruebas técnicas.
* 📡 Prácticas de redes, Linux, Cisco Packet Tracer y administración de sistemas.

Me interesa crear software útil, automatizado y bien estructurado, con especial foco en **IA aplicada, bots, backend ligero, redes y sistemas**.
