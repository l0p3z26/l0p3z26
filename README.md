## 👤 Sobre mi

Soy estudiante de **Sistemas Microinformaticos y Redes** en Madrid. Combino mi formacion tecnica en redes y sistemas con desarrollo web, bots de Discord, automatizacion e integracion de IA generativa.

En este perfil de GitHub publico y desarrollo:

- 🎨 Prototipos web y portfolios con enfoque responsive.
- 🤖 Aplicaciones con IA, APIs externas y automatizacion.
- 💬 Bots de Discord con Python, SQLite, moderacion, tickets, logs y arquitectura modular.
- 🛠️ Scripts y herramientas para productividad, sistemas y pruebas tecnicas.
- 📡 Practicas de redes, Linux, Cisco Packet Tracer y administracion de sistemas.

Me interesa crear software util, automatizado y bien estructurado, con especial foco en **IA aplicada, bots, backend ligero, redes y sistemas**.
